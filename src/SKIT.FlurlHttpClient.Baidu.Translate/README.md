基于 `Flurl.Http` 的百度翻译开放平台 HTTP API SDK。

---

### 【功能特性】

-   基于百度翻译开放平台推送 API 封装。
-   提供了百度翻译开放平台推送 API 所需的 MD5、HMAC-SHA256 等算法工具类。
